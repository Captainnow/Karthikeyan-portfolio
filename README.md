# Professional Portfolio

Link - https://karthikeyan-portfolio-ten.vercel.app/  

A modern, responsive professional portfolio website designed to showcase multidisciplinary expertise in data analytics, AI, biomedical engineering, and system design.

## Features
- Clean, editorial aesthetic with high-impact visual layout
- Smooth animations and transitions for an engaging user experience
- Responsive design tailored for all devices (desktop, tablet, mobile)
- Dedicated sections for Experience, Projects, Skills, and Contact
- Interactive project showcase (e.g., Global Tech Layoffs Dashboard)

## Technologies Used
- HTML5
- CSS3 (Custom styling and animations)
- JavaScript

## Setup & Deployment
1. Clone the repository.
2. Open `index.html` in any modern web browser to view the portfolio.
3. No build steps or local server required for core functionality.
